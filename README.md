# JournaLog

## 日々の積み上げを可視化し、継続を支援するジャーナルアプリ

## 背景・課題
- 自己啓発系Youtuberである-ジョージ-さんの影響で、ジャーナルを書く習慣を始めた
- ジャーナルでは、目標やその日の行動、振り返りをノートに記録していた
- しかし、ノートに手書きで書く作業が次第に面倒に感じるようになった
- 電車などの移動中は混雑しており、ノートを開いて記録することが難しかった
- また、ノートは使い切るたびに買い直す必要があり、過去の記録を振り返りづらいという課題があった
- これらの記録をPCやスマートフォンから、いつでもどこでも入力・確認できたら便利だと感じた

## 解決策 / コンセプト
ノートの代わりに、Goal・Todo・スケジュール・振り返りの4つの要素を1つのJournalとして管理できるアプリを作成しました。
この4つを組み合わせることで、「目標 → 行動 → 振り返り」という流れを自然に作れる設計にしています。
これにより、ジャーナル習慣をPCやスマートフォンから無理なく継続できるようにしました。

## 主な機能
- ジャーナルを記録できる
- 1日を通して達成したい目標(Goal)、目標に対して何を行えばいいか(Todo)、その日のスケジュール(Schedule)、その日の出来事の振り返り(Reflection)の4つを記録できます。
- アカウントを作りログインするとこれらの機能を使うことができるようになります(ログイン必須)。

## 説明
1. ログインを行います。(アカウントを持っていない方はサインアップ。)
2. カレンダーが表示されるので記録したい日付を選択してください。
3. Goal(現在の自分の目標)とTodo(Goalを踏まえた今日やるべきこと)を設定し、保存してください。
4. 保存後の画面で下にスクロールするとSchedule(時間割)の出てくるのでスケジュール管理をしたい場合はタイトルと時刻を入力してください。
5. さらに下に行くとReflection(振り返り)があるので任意記入。

## 使用技術
- Python 3.12
- Django 6.0
- SQLite3(MySQLに移行する予定)
- Bootstrap 5
